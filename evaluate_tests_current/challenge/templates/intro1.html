Introduction

{% endblock %} {% block form_content %}

For this challenge, you will be presented with a series of code snippets written in C. Each piece of code will contain a number of print statements that, when executed, print text to the screen. An example looks like this:

void func_one(int input) {
    printf("1 ");

    if(input > 5) {
        printf("2 ");

    } else {
        func_two();
    }
}

void func_two() {
    printf("3 ");
}

You will be given a text box where you can provide different inputs for func_one. Each line consists of the inputs necessary for a single execution of func_one. Because func_one takes an int, each line should consist of a single integer. Some example inputs are given. You can change the inputs and/or see what they would print to the screen when func_one is executed.
